Резюме
------

Протокол **HTTP/2** это:

* двунаправленный (запрос/ответ, Push)
* бинарный протокол
* совместим с HTTP/1.1
* не хранит состояния
* работает на сетевом уровне только через **TCP**
* позволяет передавать все ресурсы в одном соединении (мультиплексирование)
